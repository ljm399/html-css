# 网易云音乐官网重构

## 📖 项目简介

这是一个**网易云音乐官网的前端静态页面重构项目**，完整复刻了网易云音乐 PC 端官网的首页，包括顶部导航栏、轮播图、推荐内容、排行榜等核心模块。项目注重还原官网 UI 细节，实现了像素级的页面布局和交互效果。

### 首页效果

## ✨ 核心功能

- **顶部导航栏**：包含 Logo、导航菜单、搜索框、用户登录入口
- **轮播图模块**：首页焦点图自动轮播展示
- **推荐内容**：热门歌单、新碟上架、榜单推荐
- **精美布局**：多列布局、卡片式设计
- **响应式交互**：hover 效果、图片加载优化
- **SEO 优化**：语义化 HTML 标签，h1 标签文字隐藏优化

## 🛠️ 技术栈

### 核心技术

- **HTML5**：语义化标签（header、nav、section、footer 等）
- **CSS3**：Flexbox 布局、定位、伪元素、过渡动画
- **CSS Sprites**：雪碧图技术优化图片加载性能

### 技术亮点

- **模块化 CSS**：将样式拆分为多个 CSS 文件（topbar、carousel、main 等）
- **CSS Reset**：消除浏览器默认样式差异
- **Sprite 图技术**：合并小图标减少 HTTP 请求
- **BEM 命名规范**：清晰的 CSS 类名结构
- **语义化 HTML**：利于 SEO 和可访问性

## 🎯 技术要点

### 1. 布局技术

- 使用 **浮动（float）** 和 **定位（position）** 实现复杂布局
- 采用 **wrapper 容器** 实现页面居中
- **双栏布局** 和 **多栏布局** 的实践

### 2. CSS Sprites

```css
.sprite_img {
  background: url(../img/sprites.png) no-repeat;
}
```

- 将多个小图标合并为一张雪碧图
- 使用 background-position 定位显示不同图标
- 减少 HTTP 请求，提升页面加载速度

### 3. SEO 优化

```html
<h1 class="logo sprite_img">
  <a href="#">网易云音乐</a>
</h1>
```

- 使用 h1 标签包含网站名称
- 文字内容利于 SEO，通过 CSS 隐藏实现视觉效果

### 4. 样式模块化

- 按功能模块拆分 CSS 文件
- 便于维护和协作开发
- 清晰的代码结构
